Palavras reservadas



As palavras reservadas, também conhecidas como palavras chave da linguagem C, são comandos de uso muito específico dentro da linguagem.

Estas palavras não podem ser usadas com outro propósito além do original dentro de programas em C. Por exemplo, você não pode usar palavras reservadas para dar nome a variáveis ou funções.

O padrão ANSI C definiu um conjunto de 32 palavras reservadas na linguagem C, veremos a seguir o significado de cada uma delas.

Palavra reservada: asm
Indica instruções em assembly.

Exemplo:
asm [volatile] ( "SEU CODIGO\n\t"
"EM\n\t"
"ASSEMBLY"
: OperadoresDeSaída
[ : OperadoresDeEntrada
[ : Clobbers ] ])
Palavra reservada: auto
Define a classe de armazenamento padrão, ou seja , a variável é criada quando a função é invocada e depois é destruída quando a função termina.

Uma vez atribuído um valor para uma variável auto, é impossível mudar o tipo da variável
Palavra reservada: break
Comando para forçar a saída imediata dos comandos switch, for, while, e também, do…while.


Palavra reservada: case

Comando utilizado dentro do comando switch para selecionar uma constante.

Palavra reservada: char

Tipo de dado utilizado para armazenar caracteres.

Exemplo:
Para guardar a letra C, escrevemos:

char letra = 'C';

A sintaxe linguagem de programação é case sensitive, ou seja, minúsculo é diferente de maiúsculo. Por exemplo:

char letraMinuscula = 'c';
char letraMaiuscula = 'C';

Esses dois caracteres, embora representem a mesma letra, são totalmente diferentes, pois uma é maiúscula e a outra é minúscula.

Palavra reservada: const

Impede que uma variável seja modificada, criando assim uma constante.

Exemplo:

const int aposentadoria = 65;

const int maioridade = 18;



Palavra reservada: continue

Força a interrupção dos loops:

for,
while, ou
do…while

fazendo com que passem para a próxima iteração.

Palavra reservada: default

É utilizado dentro de switch…case para tratar valores não definidos anteriormente nas opções case.

Palavra reservada: do

Estutura de repetição usada em conjunto com o while.

O comando do…while faz com que os comandos do bloco a ser repetido sejam executados no mínimo uma vez.



Palavra reservada: double

Tipo de dados numérico de ponto flutuante com precisão dupla.

Palavra reservada: else

Indica um bloco de comandos a ser executado quando a condição do comando if for falsa.

Palavra reservada: enum

Tipo de dados definido pelo programador que permite a definição de constantes.



Palavra reservada: extern

Indica que uma variável foi declarada em outro arquivo.

Palavra reservada: float

Tipo de dados de ponto flutuante com precisão simples.

Palavra reservada: for

Estrutura de repetição que utiliza condições e contador.



Palavra reservada: goto

Comando que permite “saltar” incondicionalmente para uma posição do programa pré-deteminada.

Palavra reservada: if

Comando condicional que altera o fluxo do programa de acordo com uma condição que pode ser verdadeira ou falsa.

Palavra reservada: int

É o tipo de dados mais usado para armazenar valores inteiros.



Palavra reservada: long

É um tipo de dados usado para armazenar valores inteiros que possui precisão superior ao tipo int.

Palavra reservada: register

Solicita o armazenamento diretamente nos registradores do processador.

Palavra reservada: return

Retorna um valor dentro de uma função ou força o abandono da mesma.



Palavra reservada: short

É o chamado inteiro curto.

Armazena valores inteiros com precisão menor (apenas 2 bytes) do que o tipo int.

Palavra reservada: signed

Sinaliza que uma variável pode armazenar tanto valores positivos como negativos.

Palavra reservada: sizeof

Comando usado para retornar o tamanho em bytes de um determinado tipo de dados.



Palavra reservada: static

Comando que avisa ao compilador para reter o valor de uma determinada variável.

Palavra reservada: struct

Cria as chamadas estruturas ou seja registros.

Palavra reservada: switch

Comando de seleção usando em conjunto com o comando case, permite escolher entre várias opções.



Palavra reservada: typedef

É um comando modificador usado para criar novos nomes para tipos de dados.

Palavra reservada: union

Permite que um conjunto de variáveis partilhem o mesmo espaço na memória.

Palavra reservada: unsigned

Faz com que a variável apenas aceite valores positivos ou zero.



Palavra reservada: void

Comando que indica que a função não retorna nada ou que não tem parâmetros de entrada.

Palavra reservada: volatile

Indica que uma variável pode ser alterada.

Palavra reservada: while

Estrutura de repetição que executa enquanto uma condição é verdadeira.



Observação: Todas as palavras reservadas da linguagem de programação C são sempre escritas em letras minúsculas
