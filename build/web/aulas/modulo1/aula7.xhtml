Agora vamos analisar por partes a estrutura do código que foi desenvolvido.

#include: É utilizado para incluir alguma biblioteca no programa.
Através dessas bibliotecas você podera utilizar funções como por exemplo scanf e printf sem que você tenha que implementar o código por trás delas.

<stdio.h>: Nesta biblioteca existem declarações de funções úteis para entrada e saída de dados (std = standard, padrão em inglês; io = Input/Output, entrada e saída ==> stdio = Entrada e saída padronizadas).
Toda vez que você quiser usar uma destas funções deve-se incluir este comando. O C possui diversos Arquivos-cabeçalho. Nesse programa foi utilizado apenas as funções printf, definida no pacote stdio.h. A função printf serve para imprimir uma mensagem de texto na tela do computador. Você deve entrar com os dados através do teclado (terminando com um ENTER), e esses são passados a função scanf.

/* exemplo de conteudo */: O compilador C desconsidera qualquer comando que esteja começando com /* e terminando com */, tornando o mesmo um comentário. Os comentários são úteis dentro de um programa para que a compreenção do código se torne mais fácil.

int main(): Indica que estamos definindo uma função de nome main.
Todos os programas em C têm que ter uma função main, pois é esta função que será chamada quando o programa for executado. O conteúdo da função é delimitado por chaves { }.
O código que estiver dentro das chaves será executado seqüencialmente quando a função for chamada.
A palavra int indica que esta função retorna um inteiro.


printf: É uma das funções presentes dentro da biblioteca stdio.h.
Está função é utilizada para imprimir na tela o conteudo presente dentro dos parenteses e aspas, como o utilizado no exemplo ("Ola! Eu estou vivo!\n").
O comando \n utilizado no final da frase não é exibido na tela, já que sua função é utilizada para realizar quebra de linha.
É importante observar também que os comandos do C terminam com ;

scanf: interrompe o programa e espera que o usuário digite dados de entradas.
Os dados de entrada devem ser terminados com a digitação da tecla enter. A cada carctere de controle deve corresponder uma variável a qual terá o valor que for digitado pelo usuário.
A lista de variáveis é uma lista de identificadores de variáveis separadas por vírgulas. Cada identificador deve ser precedido pelo caractere &, também conhecido como o "e comercial".

Exemplos de leituras:

Espera que o usuário digite um inteiro. O valor digitado será o conteúdo da variável n: 

 scanf("%d", &n);

Espera que o usuário digite dois inteiros. O primeiro valor digitado será o conteúdo da variável m e o segundo valor será o conteúdo da variável n. 

scanf("%d %d", &m, &n);

O usuário deve digitar n números. Note que o printf tem como finalidade somente orientar o usuário para a digitação dos números.

Neste programa foram declaradas duas variáveis chamadas Dias e Anos. A primeira é um int (inteiro) e a segunda um float (ponto flutuante). Foi realizada então uma chamada da função printf(), que coloca uma mensagem na tela.

O programa posteriormente ficou aguardando o dado que será fornecido pelo usuário para armazená-lo na variável Dias. Para tanto usamos a função scanf(). A string "%d" diz para a função que iremos ler um inteiro.

O segundo parâmetro passado para a função diz que o dado lido deverá ser armazenado na variável Dias. É importante ressaltar a necessidade de se colocar um & antes do nome da variável a ser lida quando se usa a função scanf(). Observe que, no C, quando temos mais de um parâmetro para uma função, eles serão separados por vírgula.

Temos então uma expressão matemática simples que atribui a variável Anos o valor de Dias dividido por 365.25. Como Anos é uma variável float o compilador fará uma conversão automática entre os tipos das variáveis.

A segunda chamada da função printf() tem três argumentos. A string "\n\n%d dias equivalem a %f anos.\n" diz para a função que ela deve dar dois retornos de carro (passar para a próxima linha e depois para a subsequente), colocar um inteiro na tela, colocar a mensagem " dias equivalem a ", colocar um valor float na tela, colocar a mensagem " anos." e dar mais um retorno de carro. Os outros parâmetros são as variáveis das quais devem ser lidos os valores do inteiro e do float, respectivamente.
