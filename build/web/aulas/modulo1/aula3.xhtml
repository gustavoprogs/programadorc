Variáveis



O que é uma variável?
Uma variável é utilizada dentro do compilador para determinar uma “região de memória (do computador) previamente identificada (nome da variável), cuja finalidade é armazenar os dados ou informações de um programa por um determinado espaço de tempo.


Nomes de variáveis:
É uma prática tradicional do C, usar letras minúsculas para nomes de variáveis e maiúsculas para nomes de constantes. Isto facilita na hora da leitura do código;
Quando se escreve código usando nomes de variáveis em português, evita-se possíveis conflitos com nomes de rotinas encontrados nas diversas bibliotecas que são em sua maioria absoluta, palavras em inglês.

Observações
• O nome das variáveis pode ser qualque palavra que não seja uma palavra chave da linguagem (Aula 2).
• E possivel conter um número na palavra: Casa1
• Não é aceitável iniciar com um número: 1casa (errado)
• E possivel utilizar subscrito, desde que não inicie com ele: Casa_da_ana
• Não é possível utilizar no nome da variável os seguintes caracteres:
{ ( + - * / ; . , ?



Sintaxe de declarações de variáveis em C:
Em C devemos listar primeiro o tipo, depois o nome da variável.


Pseudocódigo:
<tipo> <nome_da_variável>
